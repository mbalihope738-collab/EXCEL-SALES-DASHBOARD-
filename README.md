# EXCEL-SALES-DASHBOARD-

This project presents an interactive sales analysis dashboard built using the Superstore dataset in Microsoft Excel. The aim of the project is to demonstrate how data can be transformed into actionable business insights through structured analysis, data visualization, and dashboard design.
The dashboard provides a comprehensive overview of key performance metrics, including total sales, profit, quantity sold, and regional performance. It allows users to explore trends across different dimensions such as product categories, sub-categories, customer segments, and geographic regions. By using pivot tables, slicers, and dynamic charts, the dashboard enables real-time interaction and filtering, making it easy to identify patterns and performance drivers.
Key insights derived from the analysis include identifying high-performing product categories, underperforming regions, and areas where profitability can be improved despite strong sales volumes. The project also highlights the relationship between sales and profit, helping to uncover cases where high revenue does not necessarily translate into high profitability.
From a technical perspective, this project showcases strong Excel skills, including data cleaning, data transformation, pivot table creation, and the development of interactive dashboards. Emphasis was placed on creating a user-friendly layout with clear visuals to support decision-making for non-technical stakeholders.
This project is particularly relevant for small to medium-sized businesses looking to better understand their sales performance and optimize their strategies using data-driven insights. It reflects a practical approach to business intelligence using accessible tools.
Overall, this dashboard demonstrates my ability to turn raw data into meaningful insights and present them in a clear, interactive, and visually engaging way.
